Change labels in the portal

All the text in that portal that isn’t in your topics or the table of contents are labels. Updating the text requires updating a label file.
  1. In you favorite UTF-8 aware text editor, open the file i18n/lang.yml where lang represents the language or language-locale combo for the labels you are trying to change.
  2. Edit the label you want to change.

    Note that the labels are displayed using the r18n ruby library - see http://r18n.rubyforge.org/ for more information.

    The labels are in yaml format - see http://www.yaml.org/spec/1.2/spec.html for more information.

    Note

    Use double quotes to escape special characters.

  3. Restart your server to see the label changes.
