Replace sample content with your own

Replace the sample content with your own by removing the files in content/en-us/sample, then adding your own content in a language specific subdirectory of content.
  1. Delete content/en-us/sample
  2. Delete the existing content in the database.
    • If you are using a local database, issue the command rake destroy_local
    • If you are using a remote database, issue the command rake destroy_remote
  3. Copy your content into a language appropriate subdirectory of content. For example, if your content is in Japanese, and itâ€™s testing content, copy it into content/ja-jp/testing.
  4. Push your content to the server.
    • If you are using a local database, issue the command rake update_local_db
    • If you are using a remote database, issue the command rake update_remote_db
Your content should now be available in the portal.
