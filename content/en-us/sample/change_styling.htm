Change the styling of the portal

Changing the styling of the portal requires you to add Twitter Bootstrap and Fontawesome as git submodules. Then you update the styles by editing LESS files, which are then compiled to CSS.
  1. At the command-line, run rake bootstrap:init
  2. At the command-line, run rake bootstrap:make Running this command tests that your system is configured properly.
  3. Update the files in in the less directory.
  4. At the command-line, run rake bootstrap:make
If you are running the portal using the start_local_debug target, you will see the results immediately; if not, you’ll need to restart your server. If you are using a remote server, you’ll need to update your git repository with the new files.
