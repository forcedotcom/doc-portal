Creating a Table of Contents

Creating a table of contents for the portal is easy, but requires you follow a few rules.
  • The entire toc file is composed of a <div> element, with the class attribute set to sidebar-nav and the id attribute set to side-menu
  • Within that div, you need a child <div> element, with the class attribute set to accordian and the id attribute set to toc
  • Each group is defined by a <div> with the class set to accordion-group and the id attribute set to a unique id string.
  • Within each group is a heading. It is defined by a <div> with the class set to accordion-heading accordion-toggle
  • Within that div, you need two <a> element. The first is for the text. The second is for the +/- icon. The second must have its href set to #subnav-[accordion-group-id]

For example: 

    <div class="sidebar-nav" id="side-menu">
      <div class="accordion" id="toc">
        <div class="accordion-group" id="d2e38">
            <div class="accordion-heading accordion-toggle toc-level-1">
                <a class="toc-a-block" href="welcome.htm">
                    <span class="toc-text">Welcome</span>
                </a>
                <a data-toggle="collapse" class="pull-right toc-plus-block"
                    data-parent="#toc" href="#subnav-d2e38">
                    <i class="icon-plus toc-icon pull-right"></i>
                </a>
            </div>
  • As a sibling to the accordion-heading is an <div> with the class set to accordion-body and another unique id attribute.
  • Within each accordion-body is a div, with a child a element. The href points to the appropriate topic.
  • Repeat as needed.
